# Building #            {#building}
Build the executables use `cmake` and `make`
~~~~~~~~~~~~~
mkdir build
cd build
cmake ..
make
~~~~~~~~~~~~~

Executables will be placed in the bin folder.

## Documentation ## {#buildingDoc}
To generate the documentation run
~~~~~~~~~~~~~
make doc
~~~~~~~~~~~~~
